Working with Words

Use the <p> tag to make paragraphs.

Use the <h1> to <h6> tag to make different sized headings

Use the <strong> tag to make words bold.

Use the <em> tag to italize words.

Use the <!-- --> tag to write comments that will not show up on the webpage.

Making Lists

Things in lists are always inside the <li> tag.

To make unordered lists, use the <ul> tag, unordered lists are listed under bullet points.

  • Just
  • Like
  • So

Ordered lists are used when the order matters, for this we use the <ol> tag, ordered lists are numbered.

  1. Fuecoco
  2. Quaxly
  3. Grass Cat

Making Links

To create links, use the <a href="link"> tag.

Embedding Images

To insert images into your webpage, use the <img src="source"> tag.
